# One Thing Theme

A minimalist, distraction-free theme for JetBrains IDEs. Focus on what matters.

---

## Overview

One Thing Theme brings a clean, monochromatic aesthetic to your IDE. Inspired by the philosophy of focusing on "one thing" at a time, this theme removes visual noise and lets your code take center stage.

Inspired by the [One Thing theme for Zed](https://github.com/rust-lang-top/one-thing-theme-zed).

---

## Variants

### One Thing Dark
A deep, near-black background (#02010A) that's easy on the eyes during long coding sessions. Perfect for low-light environments and reducing eye strain.

### One Thing Light
A clean, bright background (#FAFAFA) that works great in well-lit environments. Crisp and minimal without being harsh.

---

## Features

- **Minimal & Focused** - Grayscale syntax highlighting removes color distractions while maintaining code readability
- **Carefully Crafted Contrast** - Each element is designed for optimal visibility without visual noise
- **Complete IDE Coverage** - Consistent theming across editor, tool windows, dialogs, and all UI components
- **VCS Integration** - Subtle color indicators for git changes (added, modified, deleted)
- **Error Highlighting** - Clear but non-intrusive error, warning, and info indicators
- **Language Support** - Optimized for Java, Kotlin, JavaScript, TypeScript, Python, Rust, Go, JSON, YAML, Markdown, SQL, and more

---

## Philosophy

Most themes use a rainbow of colors for syntax highlighting. One Thing takes a different approach: **less is more**.

By using primarily grayscale tones, your eyes aren't constantly jumping between different hues. Instead, subtle variations in brightness guide you through your code naturally. Important elements stand out, while supporting syntax fades into the background.

The result? A calmer, more focused coding experience.

---

## Installation

1. Open **Settings/Preferences** → **Plugins** → **Marketplace**
2. Search for "One Thing Theme"
3. Click **Install** and restart your IDE
4. Go to **Settings** → **Appearance & Behavior** → **Appearance**
5. Select **One Thing Dark** or **One Thing Light** from the Theme dropdown
